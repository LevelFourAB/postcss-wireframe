# postcss-wireframe

PostCSS plugin to help with creating wireframe styles.

## Usage

```css
.wireframe-font {
	@wireframe;
}

.wireframe-background-auto {
	@wireframe auto;
}

.wireframe-background-custom {
	@wireframe #777;
}

.wireframe-outline {
	@wireframe outline;
}

.wireframe-background-and-outline {
	@wireframe auto outline;
}
```
